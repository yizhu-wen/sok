# Reviewer Audio Clips

This directory contains one short MP3 preview per setting-level distortion
condition. The manifest is grouped into digital-level, physical-level, and
AI-induced distortion categories.

The clips are generated by:

```bash
python3 scripts/build_reviewer_audio_demo.py
```

Generation details:

- deterministic random seed: `2026`
- maximum preview duration: 10 seconds
- manifest: `demo_audio_clips/manifest.json`

The manifest records the source WAV paths used when the clips were generated.
This reviewer release keeps the playable MP3 previews in this directory.
