# Expected Open Weights

Download the upstream pretrained assets from the official FragmentVC sources
and place them in this directory with these filenames:

- `fragmentvc.pt`
- `vocoder.pt`
- `wav2vec_small.pt`

Sources:

- Official repo: https://github.com/yistLin/FragmentVC
- `wav2vec_small.pt`: https://dl.fbaipublicfiles.com/fairseq/wav2vec/wav2vec_small.pt

The packaged inference scripts assume the original filenames above.
